# CSS (Cascading Style Sheets)

This repository documents my journey of learning CSS from fundamentals.
It contains examples and practice files to understand styling, layouts, and responsiveness.

This repository is created for learning, practice, and revision.

### 4. Display & Positioning

- position (static, relative, absolute, fixed, sticky)
- z-index
- overflow

### 5. Layout Techniques
- Flexbox
- Grid (basics)

- Media queries
- Relative units (%, em, rem)
- Mobile-first approach
